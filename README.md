# QQ简单群管

AstrBot QQ 简单群管插件。

本项目面向 OneBot v11 / `aiocqhttp` 平台，提供常用 QQ 群管理、入群审核、黑名单、自动风控和可视化配置面板。插件默认不在任何群启用，需要在插件设置页或 `群管配置` 中按群开启。

## 功能

- 基础群管：禁言、解禁、踢人、总黑名单拉黑与移除。
- 消息管理：撤回消息。
- 风控：禁词检测、内置禁词、自定义禁词、刷屏检测、宵禁。
- 入群管理：进群白词/黑词、等级门槛、尝试次数、本群进群黑名单、退群黑名单、总黑名单、进群欢迎、进群禁言、退群通知、退群拉黑、主人 QQ 私信通知、批准/驳回申请。
- 群工具：群友信息。
- 配置：支持 `_conf_schema.json` 配置和 `pages/settings` 前端配置面板。

## 安装与依赖

把本仓库放到 AstrBot 的插件目录：

```bash
AstrBot/data/plugins/astrbot_plugin_qqmanager
```

AstrBot 会根据 `requirements.txt` 安装依赖：

```text
aiocqhttp
aiosqlite
apscheduler
```

## 使用

发送 `/群管帮助` 或 `群管帮助` 查看命令说明。斜杠命令仍然兼容；AstrBot 超管也可以在群内直接输入命令，不需要 `/` 前缀。

常用命令示例：

```text
禁言 600 @群友
解禁 @群友
踢 @群友
拉黑 @群友
删除总黑名单 123456
删除退群黑名单 123456
撤回
设置禁词 +词1 -词2
刷屏禁言 300
群管配置 启用插件: 开
进群审核 开
群管配置
```

`拉黑 @群友` 会把目标加入总黑名单，并立即尝试从当前群踢出。机器人随后扫描目标所在的其他群，把可踢和不可踢的群聊清单私发给 AstrBot 超管。

超管可在私聊中继续处理：

```text
是
all
1,3
/取消踢除 任务ID
```

## 配置

在 AstrBot 插件详情页打开插件 Page，可进入 `settings` 配置面板。

配置面板支持：

- 按群启用或关闭插件。
- 默认群模板。
- 按群独立配置。
- 总黑名单维护，保存后会私发超管确认是否跨群踢除新增用户。
- 群列表同步和机器人群身份展示。
- 指令权限分级管理。

说明：

- “超管”指 AstrBot 全局配置里的管理员 `admins_id`。
- “主人QQ”是本插件用于接收退群通知等管理消息的 QQ 列表，不等同于超管权限；跨群踢除确认命令只响应 AstrBot 超管私聊。
- 退群黑名单按群保存，只影响重新申请同一个群；总黑名单对所有已启用插件的群生效。

持久化数据会写入 AstrBot 插件数据目录：

```text
data/plugin_data/astrbot_plugin_qqmanager/
```
